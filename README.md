# Until Then - Basta Driver Auto-Bot

Auto-spacebar for the "Sweet Lover" achievement in Until Then's jeepney minigame. Because i suck and this is the only achievement i couldn't get.

## Requirements

- Windows
- .NET SDK or Visual Studio
- Administrator privileges

## Installation

### Option 1: Build from Source

1. Install [Visual Studio](https://visualstudio.microsoft.com/downloads/) (Community edition is free)
2. Create a new Console App (.NET Core or .NET Framework)
3. Copy the code into `Program.cs`
4. Build the project (Ctrl+Shift+B)
5. Find the `.exe` in `bin\Debug\` or `bin\Release\`

### Option 2: Using Command Line

1. Install [.NET SDK](https://dotnet.microsoft.com/download)
2. Save the code as `Program.cs`
3. Open Command Prompt in that folder
4. Run: `dotnet run`
